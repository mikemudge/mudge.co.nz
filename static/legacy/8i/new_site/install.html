System Requirements   Release Notes

The 8i Portal requires the use of SteamVR
  1. Install the latest version of Steam on your computer
  2. Open Steam and select Library > Tools > Steam VR
  3. Right click on SteamVR and select Install Game
  4. Set up your VR headset and at least 1 controller
  5. Launch SteamVR
  6. Right click VR Monitor window > Run Room Setup
  7. Follow on-screen instructions
For more information please visit
http://steamcommunity.com/app/250820
