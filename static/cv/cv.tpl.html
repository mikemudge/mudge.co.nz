8i
2015-2017

As one of the early developers at 8i I worked on a variety of projects, starting off on several solo projects before working on our AR app, Holo, with a larger team. As the company grew I was looked upon to provide history and context for the engineering teams, handing over code as necessary.

Website
I redeveloped 8i.com twice using Wagtail/Django running on Amazon cloud. The front-end started with Angular, moving to jQuery/JavaScript and HTML with CSS. I integrated Sphinx for documentation and used markdown from 8i Plugins to generate HTML. I implemented the site with a CMS which allowed for content managers to have complete control of the content, including uploading new images and creating sub models for listed content.

Web Player
I developed a web player similar to YouTube for playing our volumetric videos using three.js. This required an understanding of 3D vector mathematics.

Holo Content Platform
I worked in a team of two to build the backend and CMS for Holo. Holo had 1,000,000+ downloads and was dealing with an average of ~150 users at a time. Backend was distributed using Amazon’s ECS and EC2 using docker images for easy rollback. It used BuildKite agents to continuously test and deploy automatically from Github. Content requirements included user access controls, content grouped by character, tags and promotions. It also required content creation, including file upload to Amazon, screenshot generation, grouping/ordering and adding metadata. Also supported Draft/Private and Published content.

Director
I developed a web editor allowing creation of a scene to compliment our volumetric videos. I was responsible for the implementation and delivery, but I worked with developers, product owners and others to define a flexible, extendable scene graph schema. It supported adding 8i 3D videos, 3D models (obj format), additional audio, background photo spheres, 3D text and 2D images as a plane. Each item had a timeline of keyframes for position, rotation and scale as appropriate. 8i worked with Life magazine to create the “Buzz Aldrin: Cycling Pathways to Mars” VR experience and web director was used to create a teaser while the full VR experience was being worked on.
